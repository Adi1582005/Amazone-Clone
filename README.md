# Amazone-Clone

<b>Amazon Clone (Frontend Project)</b>

Technologies:<b>HTML5</b> , <b>CSS3</b>

<li>Built a static clone of the Amazon homepage to practice modern frontend layout techniques.</li>
<li>Implemented navigation bar, hero banner, product grid, and footer sections similar to the original website.</li>
<li>Used CSS Flexbox and Grid for responsive layout and structured design.</li>
<li>Focused on pixel-perfect UI design and clean code structure.</li>
